# 📊 Human Resource Insights Dashboard (Power BI)

---

## 📌 Overview

This project presents an interactive **Human Resource Insights Dashboard** built using Power BI. It analyzes employee data to uncover key workforce trends such as attrition, salary distribution, department performance, and hiring patterns.

The goal is to help HR teams make **data-driven decisions** and improve employee retention strategies.

---

## 🎯 Problem Statement

Organizations often face challenges in understanding:

* Why employees are leaving (attrition)
* Which departments are most affected
* How salary impacts retention
* Hiring trends over time

This dashboard addresses these problems by providing clear and interactive insights.

---

## 🛠️ Tools & Technologies

* Microsoft Power BI
* Excel (Data Source)
* DAX (Data Analysis Expressions)

---

## 📂 Dataset

* Source: Sample HR Dataset (Dummy Data)
* Size: ~50,000 records
* Key Columns:

  * Employee ID
  * Department
  * Gender
  * Salary
  * Attrition
  * Hiring Date

---

## 📊 Dashboard Features

* ✅ KPI Cards (Total Employees, Active Employees, Attrition, Avg Salary, Attrition Rate)
* ✅ Department-wise Employee Distribution
* ✅ Gender-wise Analysis
* ✅ Year-wise Hiring Trends
* ✅ Interactive Filters (Department, Gender, Year)

---

## 📈 Key Insights

* 🔹 Total Employees: **50K**
* 🔹 Active Employees: **40K**
* 🔹 Attrition: **10K (~20%)**
* 🔹 Gender distribution is nearly balanced (~50% Male / 50% Female)
* 🔹 Marketing and HR departments have the highest employee count
* 🔹 Hiring trend remained stable from 2020–2023 with a slight drop in 2024
